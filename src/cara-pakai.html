Cara Pakai

1. Tambah Gambar

Untuk memulai, langkah pertama adalah memilih gambar yang ingin Anda beri watermark. Tekan tombol "Tambah Gambar" di bawah ini.

Gambar yang ditambahkan tidak akan dikirim keluar dari perangkat yang Anda gunakan.
{% image "src/img/tambah-gambar.png", "Tombol 'Tambah Gambar'", [350, 100], "rounded-lg border border-gray-200 shadow hover:scale-105 transition" %}

2. Watermark Otomatis

Jika Anda ingin menambahkan watermark secara otomatis, centang checkbox "Watermark Otomatis".

{% image "src/img/watermark-otomatis.png", "Checkbox 'Watermark Otomatis'", [350, 100], "rounded-lg border border-gray-200 shadow hover:scale-105 transition" %}

3. Masukan Teks

Setelah itu, ketikkan teks watermark yang diinginkan dalam kotak teks. Sebagai contoh, "10-10-2021 Verifikasi Pinjaman Online".

{% image "src/img/box-teks.png", "Input watermark", [350, 100], "rounded-lg border border-gray-200 shadow hover:scale-105 transition" %}

4. Pilih Font

Pilih jenis teks yang diinginkan dengan memilihnya dari kotak "Font" seperti yang terlihat pada gambar di bawah ini.

{% image "src/img/box-font.png", "Opsi font watermark", [350, 100], "rounded-lg border border-gray-200 shadow hover:scale-105 transition" %}

5. Pilih Posisi Watermark

Pilih posisi watermark melalui kotak "Posisi Teks". Alternatifnya, Anda dapat menentukan posisinya langsung dengan menekan teks pada kotak gambar di sebelah kanan dan menariknya ke lokasi yang diinginkan.

{% image "src/img/box-posisi.png", "Opsi posisi watermark", [350, 100], "rounded-lg border border-gray-200 shadow hover:scale-105 transition" %}

6. Pilih Ukuran Watermark

Pilih ukuran watermark dari opsi yang tersedia pada kotak "Ukuran Font".

{% image "src/img/box-ukuran.png", "Opsi ukuran watermark", [350, 100], "rounded-lg border border-gray-200 shadow hover:scale-105 transition" %}

7. Tentukan Warna Watermark

Setelah itu, pilih warna watermark yang diinginkan dari kotak opsi "Warna Teks".

{% image "src/img/pilih-warna.png", "Input warna watermark", [350, 100], "rounded-lg border border-gray-200 shadow hover:scale-105 transition" %}

8. Mengatur Rotasi Watermark

Sesuaikan rotasi watermark dengan menggunakan opsi "Rotasi Teks". Geser-geser nilai rotasi untuk mendapatkan tampilan yang diinginkan.

{% image "src/img/atur-rotasi.png", "Input rotasi watermark", [350, 100], "rounded-lg border border-gray-200 shadow hover:scale-105 transition" %}

9. Mengatur Opasitas Watermark

Sesuaikan tingkat opasitas watermark dengan menggunakan opsi "Opacity Teks". Geser-geser nilai opasitas untuk mendapatkan efek yang diinginkan.

{% image "src/img/atur-opasitas.png", "Input opasitas watermark", [350, 100], "rounded-lg border border-gray-200 shadow hover:scale-105 transition" %}

10. Mereset watermark

Jika Anda perlu mengatur ulang watermark, gunakan tombol "Reset" untuk mengembalikan semua pengaturan ke nilai awal.

{% image "src/img/reset-watermark.png", "Tombol reset watermark", [200, 73], "rounded-lg border border-gray-200 shadow hover:scale-105 transition" %}

11. Download Gambar

Setelah menyesuaikan watermark sesuai keinginan Anda, tekan tombol "Download" untuk menyimpan gambar KTP yang telah di-watermark ke perangkat Anda.

{% image "src/img/download-gambar.png", "Tombol download gambar", [200, 73], "rounded-lg border border-gray-200 shadow hover:scale-105 transition" %}
