Libraries

Berikut merupakan libraries yang digunakan dalam pembuatan Watermark KTP.

html2canvas

html2canvas adalah sebuah JavaScript library yang memungkinkan pengguna untuk mengambil "screenshots" dari halaman web atau bagian tertentu secara langsung melalui peramban (browser) pengguna.

Watermark KTP memanfaatkan html2canvas pada halaman Utama Watermark KTP untuk mengekspor gambar yang telah diberi watermark.

Untuk mempelajari lebih lanjut tentang html2canvas, silakan kunjungi html2canvas.hertzen.com.

Signature Pad

Signature Pad adalah sebuah JavaScript library yang dirancang untuk menggambar tanda tangan dengan lancar.

Watermark KTP memanfaatkan Signature Pad pada halaman Tanda Tangan Online untuk menggambar tanda tangan pada canvas.

Untuk mempelajari lebih lanjut tentang Signature Pad, silakan kunjungi szimek.github.io/signature_pad.
